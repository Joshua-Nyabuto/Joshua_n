PROJECTS

1. Sales management system.

Sales Management System is the system which help businessmen to manage and monitor sales of products in one or more shops. The system is able to indicate which shop sold particular item and its dairly sales report. The system also do analysis of both available and unvailable stock.
This system reduces theft in shops by sellers since it keep track on sold, unsold items and the selling prices of the items.
I developed this system in PHP,HTML and JAVASCRIPT programming languages using MYSQL database.

2. Requisition system

Requisition system is one which can help the campanies in requisition of particular item. For example if am working in Company A and i want to request hard-drive for my computer i will just login to the system using provided login creditials, then i fill the requisition form, submit and wait for approval by relevant authority. After the requisition is approved, i can now download the form and take it to Finance department to give out the money for the purchase of my hard-drive.
I developed this system in PHP,HTML and JAVASCRIPT programming languages using MYSQL database.

3. Book cost estimation System

This is the system that can be used by publishers and printing companies to estimate the cost of book given the book size, book color and number of pages. The system calculate the cost based on machines selected for this production consindering machines rates
This help to give the customer immediently response of how much it may cost for his/her books to be produced.
I developed this system in PHP,HTML and JAVASCRIPT programming languages using MYSQL database.

4. Visitors management system.

Visitors management system is one that keep track on the visitors who visits the companies and generates the reports about the same. It runs at reception, when the visitor come, he/she gives the relevant information according to desire of the company then goes to the department of visit. Upon completing what he/she came to do, then the visit exit through the system. By doing this we eliminate the manual system that requires alot of time to analyse.
